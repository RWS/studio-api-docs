# Native Verifier

Native verifiers are an essential part of the verification framework, enabling validation of native output formats such as DOC, MIF, or XML. They ensure that translated documents meet specific criteria and maintain structural integrity.

## Learn More

For detailed guidance on creating native verifiers, refer to the [File Type Support - The Verification Framework](../filetypesupport/create_a_native_verifier_introduction.md) section.
