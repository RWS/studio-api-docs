# Bilingual Verifier

Bilingual verifiers are a key component of the verification framework, designed to validate intermediary (XLIFF) files for specific file types. They help ensure that translations adhere to predefined rules and constraints.

## Learn More

For detailed instructions on creating bilingual verifiers, refer to the [File Type Support - The Verification Framework](../filetypesupport/create_a_bilingual_verifier_introduction.md) section.
