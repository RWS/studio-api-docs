    Interface IStructureContextInfo

    Represents structure context information for a paragraph in a translatable document. This object can be used to customize how this specific structure context is displayed.

    Namespace: Sdl.TranslationStudio.Api.Editor
    Assembly: Sdl.TranslationStudio.Api.dll
    Syntax

    Remarks

    The list of available structure contexts can be retrieved though StructureContexts.
