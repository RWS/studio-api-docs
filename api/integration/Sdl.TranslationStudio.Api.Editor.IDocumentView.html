    Interface IDocumentView

    Interface for document views. A document view is a UI element that shows a certain aspect or representation of an internal document (IInternalDocument).

    Namespace: Sdl.TranslationStudio.Api.Editor
    Assembly: Sdl.TranslationStudio.Api.dll
    Syntax
    public interface IDocumentView
    Remarks

    In order to create additional document views for certain types of documents, create a document view provider (see IDocumentViewProvider).
