    Interface IDocument

    Represents a generic document in the editing environment.

    Namespace: Sdl.TranslationStudio.Api.Editor
    Assembly: Sdl.TranslationStudio.Api.dll
    Syntax
    public interface IDocument
    Remarks
    The editing environment supports internal and external documents. Internal documents are documents that are presented by a document view part within the editing environment. External document are documents that are displayed externally (for example in MS Word).
