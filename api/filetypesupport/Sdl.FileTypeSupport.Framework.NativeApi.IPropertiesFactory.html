    Interface IPropertiesFactory

    Interface implemented by factories for property objects.

    The type of the context to create.

    Returns
    Type Description
    IContextInfo
    Remarks

    If the context type is one of the values defined in Sdl.FileTypeSupport.Framework.Core.Utilities.NativeApi.StandardContextTypes or Sdl.FileTypeSupport.Framework.Core.Utilities.NativeApi.TMMatchContextTypes, then the returned context will have the properties set to the default preferred values for that context type. If a different value is used, the returned context will have empty values for all other context properties.
