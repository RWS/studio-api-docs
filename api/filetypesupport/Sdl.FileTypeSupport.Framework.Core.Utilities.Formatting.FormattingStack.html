    Class FormattingStack

    Holds a collection of formatting instances that are to be treated as layered on top of each other. The EffectiveFormatting compiles the layers into a single formatting instance that represents the accumulated effect of all the layers.

    Remarks

    This is implemented as a list rather than a stack to provide more flexibility in accessing the layers. The order of the items in the list determines their precedence. The most significant item comes first.
