    Class ObservableDictionary<TKey, TValue>

    An implementation of IDictionary<TKey><TValue> which fires System.Collections.Specialized.INotifyCollectionChanged and System.ComponentModel.INotifyPropertyChanged when required. This allows the dictionary to be used as a data source for Silverlight and WPF user controls implementing data-binding.
