GetOrCreateMetadata(String)

Retrieves the default metadata for the specified language code, or if none found, creates metadata from the corresponding System.Globalization.CultureInfo. Note: this can be used during testing, but in deployment, GetMetadata(String) should be used, with fallback behaviour to use a different language code if that returns null.
