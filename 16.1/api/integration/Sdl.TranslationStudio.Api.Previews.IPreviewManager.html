Interface IPreviewManager

Manages all the previews that are supported by all the files in all the open documents. Also explicilty tracks a SelectedPreviewSet and SelectedGenerateType that can be used as default when generating previews and interacting with a user interface.

Namespace: Sdl.TranslationStudio.Api.Previews
Assembly: Sdl.TranslationStudio.Api.dll
Syntax
public interface IPreviewManager
Remarks

Set the ActiveView and access the ActiveViewPreviewManager to manage the previews available for a specific view and its associated document. Handle the PreviewSetsChanged event to populate a list of available previews in the user interface, and set the SelectedPreviewSet when the user selects a preview set. Call GeneratePreview(GenerateType) to initiate display of the preview.

Handle the ActivePreviewChanged event to detect when a generated preview is activated, in order to display it.

The ActiveViewPreviewManager provides access to all previews associated with the currently active view, you can use this to switch between the previews available (generated and tracked) for a single view.

Call SetActivePreviewFromSelection() to activate the preview corresponding to the SelectedPreviewSet and SelectedGenerateType for the active view. The ActivePreview will be updated accordingly (or set to null if no preview has been generated for this selection.)
