Interface IViewPreviewManager

Manages the previews available for a specific ITranslatableDocumentEditingView. The View property provides access to the view that is being tracked for a specific instance. Access the ActivePreviewSets to determine which preview sets can be used for the current selection in the view. The DefaultPreviewSet property is also updated based on the current view selection and reflects the default preview set defined in the filter definition for the active file in the view.

Remarks

The implementation tracks a set of previews available for the current view. This allows (among other things) source and target previews to be tracked concurrently, which means the user can switch between them without having to re-generate the previews.

The user interface may activate a specific preview enumerating the currently managed previews exposed in ManagedPreviews and calling ActivatePreview(IManagedPreview) for one of them. If as a result the ActivePreview property is null the user interface can explicitly generate a preview by calling GeneratePreview(PreviewSetInfo, GenerateType). Once the preview is generated it becomes the active preview automatically (and the ActivePreviewChanged event is raised).

The implementation automatically takes care of setting up communication with navigable and real-time updating previews. The user interface can explicitly navigate previews to the current selection in the view by calling NavigatePreviewsToActiveViewSegment().

Once a preview is no longer needed it can be removed by calling RemovePreview(IManagedPreview). All previews can be removed by calling Clear(). Both of these methods take care of properly disconnecting the previews if they are navigable and/or real-time updating.

GeneratePreview(PreviewSetInfo, GenerateType)

Initiates generation of a preview. The preview file generation is executed asynchronously, once it is completed the preview is launched and becomes the ActivePreview, at which point the PreviewGenerated event is raised. The preview will from now on be tracked by the manager, it is available through the ManagedPreviews collection.
