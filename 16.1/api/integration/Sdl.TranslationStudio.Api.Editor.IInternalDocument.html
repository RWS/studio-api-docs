Syntax
public interface IInternalDocument : IDocument
Remarks

Internal documents are documents that are presented by a document view part within the editing environment. The rely on on or more views to represent their content, of which one is the active view at all times. The view are represented as tabs. If there is only one view, no tabs are shown.
