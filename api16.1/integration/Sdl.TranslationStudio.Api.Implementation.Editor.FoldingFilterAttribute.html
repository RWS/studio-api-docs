    InsertAfter

    This property specifies the id of another filter or header that this item will be inserted after. However, if InsertBefore is also set to an empty string ("") and InsertAfter is set to an id of a header then this item will be inserted at the end of the list of items in that header. Otherwise, if set to a an empty string ("") and InsertAfter is null then this property specifies that this item will be inserted at the start of the list.
