# Open source Var:ProductName App Store plugins

## Intro

This repository contains around 50 plugins developed for Var:ProductName. Most of the plugins were developed by RWS, but some were initially developed by external contributors. We own the source code for all these plugins under our [License agreement](https://github.com/RWS/Sdl-Community/blob/master/LICENSE).

We encourage everyone who is interested to contribute by fixing issues, adding features, or enhancing documentation. To contribute, clone the repository, make changes, and submit a [pull request](https://www.codenewbie.org/blogs/how-to-make-a-pull-request).

The repository is available at [RWS Sdl-Community GitHub](https://github.com/RWS/Sdl-Community). For additional resources and guidance, explore the [documentation](../../index.md).

If you have any questions, feel free to ask on the [RWS Language Developer Community](https://community.sdl.com/developers-more/developers/language-developers).


## We want your feedback

If you have any suggestions or find any issues, please let us know on the [RWS Language Developer Community](https://community.sdl.com/developers-more/developers/language-developers).
