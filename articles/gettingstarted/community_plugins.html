    Open source Trados Studio App Store plugins

    Intro

    This repository contains around 50 plugins developed for Trados Studio. Most of the plugins were developed by RWS, but some were initially developed by external contributors. We own the source code for all these plugins under our License agreement.

    We encourage everyone who is interested to contribute by fixing issues, adding features, or enhancing documentation. To contribute, clone the repository, make changes, and submit a pull request.

    The repository is available at RWS Sdl-Community GitHub. For additional resources and guidance, explore the documentation.

    If you have any questions, feel free to ask on the RWS Language Developer Community.

    We want your feedback

    If you have any suggestions or find any issues, please let us know on the RWS Language Developer Community.
