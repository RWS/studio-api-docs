    Plugin use cases for Trados Studio

    Create new Trados Studio features

    MSWord Grammar Checker

    This app adds grammar checking capability to SDL Trados Studio by integrating the grammar checking feature of MS Word into the editor view.

    Glossary Plugin

    As an extra feature in the ribbon, this plug-in allows users to add terminology to projects in Trados Studio with just a few clicks.

    Jobs

    The Jobs plug-in adds a new view where users can find ProZ.com translation job posts online within Studio easily and quickly using search filters.

    Access external resources

    Web Lookup!

    This app allows users to highlight a word or phrase to perform a web search on directly within Trados Studio's translation editor environment.

    Integrate with apps and products

    Post-Edit Compare

    Creating a brand new view in Trados Studio, the Post-Edit Compare app provides a full analysis of Post-Editing work in an extensive report format.
