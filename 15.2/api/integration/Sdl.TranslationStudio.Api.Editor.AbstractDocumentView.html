    Class AbstractDocumentView

    Abstract base class for document views. A document view is a UI element that shows a certain aspect or representation of an internal document (IInternalDocument).

    public abstract class AbstractDocumentView : IDocumentView, IDisposable
    Remarks

    In order to create additional document views for certain types of documents, create a document view provider (see IDocumentViewProvider).

    For translatable document views, extend AbstractTranslatableDocumentView and optionally implement ITranslatableDocumentEditingView.

    Dispose()

    Called by the document when it is being disposed. Derived classes should not override this method, but rather Dispose(Boolean), which will also be called from the finalizer implementation.

    Declaration
    public virtual void Dispose()
    Remarks

    The default implementation calls Dispose(true) and then sets GC.SuppressFinalize(this) in order to avoid executing the finalizer.

    Dispose(Boolean)

    Called by: The document when it is being disposed, The IDisposable implementation, The finalizer.

    Declaration
    protected virtual void Dispose(bool disposing)
    Parameters
    Type Name Description
    System.Boolean disposing

    indicates whether the call originates from the finalizer or not. When called from the finalizer any managed references should not need to be explicitly disposed
    Remarks

    Derived classes should dispose of managed and unmanaged resurces here.

    Managed resources do not need to be disposed when called from the finalizer, i.e. when disposing is false.

    The default implementation does nothing.
