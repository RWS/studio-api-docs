    public class GenericVerifierSettings : AbstractSettingsClass, INotifyPropertyChanged, ISettingsClass, ICloneable, IEquatable<ISettingsClass>
    Remarks

    This settings class supersedes the GenericTagSettings class, which is only kept for backward compatibility with file types that don't yet use the new WPF/Silverlight UI and settings storage mechanism for tag verification settings.
