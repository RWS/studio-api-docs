    WhitespaceCharacters

    Contains a set of known whitespace characters from different languages. This should be equivalent to the set of characters for which System.Char.IsWhiteSpace(System.Char) returns true, which however is incomplete in earlier .Net versions.

    The elements of the array are sorted in ascending order, so binary search can be used.
