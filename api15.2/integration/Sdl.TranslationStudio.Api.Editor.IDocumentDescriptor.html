    IsSameDocument(String)

    Returns true if the file specified in filePath potentially represents the same document as the file represented by the specified document descriptor. This should take into account potential changes in file path of teh specified descriptor, which could lead to the two documents becoming equal and therefore being saved into the same location. The editor service will call this method when the user attempts to open a document to ensure that two such documents cannot be opened at the same time, which could lead to data loss.
